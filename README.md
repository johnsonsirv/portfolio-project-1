### Built with:

- HTML
- CSS & CSS Grids
- npm

### Installation:

Clone the repository to your local machine

> Open the terminal

```sh
$  git clone https://github.com/johnsonsirv/portfolio-project-1.git
```

### Usage

> Go to the directory

```sh
$ cd portfolio-project-1
$ npm install
$ npm run start
```

> This will launch http://localhost:8080/
